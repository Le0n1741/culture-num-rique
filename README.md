# Projet AlgoBox

Ce dépôt contient un algorithme réalisé avec le logiciel AlgoBox dans le cadre du projet.

## Contenu
- Dossier `src/` : fichiers AlgoBox (.algo)

## Prérequis
- Logiciel AlgoBox installé sur l'ordinateur

## Exécution
1. Ouvrir le logiciel AlgoBox
2. Ouvrir un fichier `.algo` depuis le dossier `src/`
3. Cliquer sur "Exécuter" pour lancer l'algorithme
